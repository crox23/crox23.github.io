Code challege journal #1

There are many path that we can use for our img in the html file. Here are some useful info:

  • If the picture is in the same folder: <img src="picture.jpg">
  • If you create a folder "images" where to insert the picture: <img src="images/picture.jpg">
  • If the picture is one level upper from the current location: <img src="../picture.jpg">
More things that I have found out while I was coding this list:
  • If you want insert a sample of code in the html file you must use "<xmp>code sample"
  • and Google and Stackover flow are really your friend
  • The link btw pages like mother/son are case sensitive! My "hello_world.html"page was not loading
